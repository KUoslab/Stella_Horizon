SLO config description

VM name: Name of VM at Horizon dashboard
  (e.g. Stella_1, Stella_2, Stella_3 eta.)
SLO option: Select SLO option
  (e.g. n_maxcredit, b_bw, c_usage)
SLO value: define specific SLO requirement
  (Numbers are acceptable. However for c_usage allows between 1 to 100)

SLO config example

  (Set 10,000 pps (packet per sec) for Stella_1 SLO)

  (Set 600(MB/s) for Stella_2 SLO)

  (Set 30% of vcpu per core for Stella_3 SLO)
